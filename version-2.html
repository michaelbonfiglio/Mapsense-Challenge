Michael Bonfiglio Mapsense challenge

I noticed the tables don’t all always show up for each project. These were the 3 different configurations that I saw for it.

Design 2

This solution has the map sets appear as rows that expand like a drawer when you click on the set. Organized as a table, it is more reserved than the first option, but if users are people who work with data a lot, I think this provides an immediate visual corollary to a spreadsheet, i.e. something they are familiar with. It will visually scale well, too. I imagined that you would only be able to have one data set open at once, so if you have one open and click another, the first collapses.

Mobile

This is similar to the previous design solution of separating the pages. I think with this design there is the possibility of having the accordion work on mobile, however, I think going to a separate page is cleaner and more action based.
